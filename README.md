#　Dokkyo-Info. WEB Application ver.0.0

## 概要
このWEBアプリはNext.jsを用いて構築されている。

・フォントの選定、導入
・スマホ環境での下部のメニュー表示
・詳細ページのSNS、WEBサイトへのボタン作成
・Font awesomeの導入と調整
・最終的なUI調整
・Google Analytics 導入
・個人情報に関する保護の指針策定、表示
